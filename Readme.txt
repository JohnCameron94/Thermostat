Thermostat Project

Description: 
  -Personal Project i created using an arduino uno
  
Purpose: 
  Create a thermostat using a LCD display, thermometer and a push button
  
The temperature is displayed for the user on the lcd display, if the user wishes to change the 
temp unit, the program offers Celcius and Fahrenheit. The user must hold down on the push button
for 5 seconds, the countdown is displayed on the lcd display. Once the 5 seconds is up, the default 
unit of temperature will change from Celcius to Fahrenheit or vice versa.
